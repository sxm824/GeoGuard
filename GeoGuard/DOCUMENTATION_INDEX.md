### I want to...

#### ...understand how signup works
1. Read **`QUICK_START_ANSWER.md`**
2. Look at **`USER_FLOW_DIAGRAM.md`**
3. Check **`Views/CompanyRegistrationView.swift`** and **`Views/SignupView.swift`**

#### ...deploy the app
1. Read **`SETUP_CHECKLIST.md`**
2. Deploy security rules (see **`FIREBASE_RULES_SETUP.md`**)
3. Set up Google Places (see **`ADDRESS_SETUP.md`**)
4. Deploy Cloud Functions (see **`functions_example.js`**)

#### ...add a new feature
1. Understand architecture (**`MULTI_TENANT_GUIDE.md`**)
2. Add to data model (`Models/`)
3. Add business logic (`Services/`)
4. Create UI (`Views/`)
5. Update security rules (**`firestore.rules`**)

#### ...modify user roles
1. Edit **`Models/UserRole.swift`**
2. Update security rules in **`firestore.rules`**
3. Update UI permissions in Views
4. Redeploy rules (**`FIREBASE_RULES_SETUP.md`**)

#### ...test the multi-tenant isolation
1. Follow testing guide in **`SETUP_CHECKLIST.md`**
2. Create 2 companies
3. Verify data isolation
4. Check security rules in Firebase Console

#### ...help a user who can't sign up
1. Check **`USER_ONBOARDING_GUIDE.md`** FAQ section
2. Check **`USER_FLOW_DIAGRAM.md`** troubleshooting
3. Verify invitation code validity
4. Check security rules deployment
